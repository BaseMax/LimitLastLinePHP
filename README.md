# Limit Last Line PHP

A tiny script to limit and remove the last line from input string using **PHP**.

## Sample

#### Input

```
1
2
3
4
5
6
```

#### Output (Remove one line from end of the string)

```
1
2
3
4
5
```

#### Output (Remove two line from end of the string)

```
1
2
3
4
```
